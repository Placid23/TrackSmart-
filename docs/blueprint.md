# **App Name**: TrackSmart+

## Core Features:

- User Profile Management: Secure user sign-up with local storage. The user provides Full name, Student identification number (optional), Monthly allowance amount, and Meal Plan Selection. This information is stored locally and used throughout the system.
- Automated Expense Tracking: Automatically log expenses based on transaction history, including transaction amount, vendor category, coupon usage (if applicable), cash usage, date, and time.
- Vendor Marketplace: Simulate a vendor marketplace with categories like School Cafeteria, Private Food Vendors, Gadget Vendors, Health & Utility Vendors, each containing predefined items with fixed prices.
- Coupon management: Manage daily cafeteria coupons allocation (Two-meal plan: ₦4,000; Three-meal plan: ₦6,000), with a system in charge of automatically invalidating it after 24 hrs. Coupons are usable only at the school cafeteria.
- Automated Insights & Decision Support Tool: Utilize a decision tree analysis to provide personalized spending advice based on tracked expenses. Generate insights and adaptive recommendations that serve the user. Classify spending as Good, Moderate, or Poor.
- Visualization Dashboard: Display daily expenses in a calendar format and provide monthly and yearly summaries with color-coded indicators for budget compliance. Show coupon usage statistics.

## Style Guidelines:

- Primary color: #3252A3 (HSL 225, 55%, 32%) symbolizing trust and stability.
- Background color: #F0F2F7 (HSL 225, 20%, 95%) providing a calm and neutral backdrop.
- Accent color: #42A7A5 (HSL 195, 55%, 52%) chosen for emphasis and interaction.
- Success Indicator: #16A34A (Green)
- Warning Indicator: #DC2626 (Red)
- Headline font: 'Space Grotesk', used for titles and headings.
- Body font: 'Inter', used for paragraphs and labels. Provides a modern, tech-forward yet readable interface.
- Use minimalist line icons to represent different expense categories, vendor types, and dashboard actions.
- Implement a clean and intuitive layout with a top navigation bar (Profile, Coupon Wallet, Notifications) and a bottom navigation bar (Dashboard, Stores, Reports) for easy access to key features. Supports intuitive navigation and mobile responsiveness.
- Add subtle hover effects, smooth page transitions, animated feedback on successful transactions, and gentle color transitions for budget alerts. Animations are kept minimal to maintain performance and clarity.
